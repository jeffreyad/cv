# Archive

This directory contains previous versions of the CV for reference.

## Naming Convention

Use the following naming convention for archived files:
- `cv-YYYY-MM.pdf` - e.g., `cv-2024-01.pdf`
- `cv-YYYY-MM.docx` - e.g., `cv-2024-01.docx`
- `cv-YYYY-MM.html` - e.g., `cv-2024-01.html`

## When to Archive

Consider creating an archive copy when:
- Starting a new job or role
- Making significant structural changes
- Adding major new sections
- Before a major application or submission
- At the end of each year

## Usage

```bash
# Example: Archive current version before making changes
cp docs/cv.pdf archive/cv-2024-12.pdf
```
